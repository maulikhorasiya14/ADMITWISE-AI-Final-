# Excluded Sources and Claims

Aggregator sites including Shiksha, Careers360, Collegedunia, CollegeDekho, College Pravesh, coaching blogs and copied tables were excluded from verified factual data. Anonymous accusations, one-line rumours, private-group content and unsupported package claims were excluded. Search-level student sources are retained only as labelled qualitative leads. No 2026 cutoff, scholarship guarantee, recruiter role or current hostel rule was invented.
