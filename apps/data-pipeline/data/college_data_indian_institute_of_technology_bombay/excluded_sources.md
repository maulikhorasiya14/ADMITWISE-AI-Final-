# Excluded Sources and Claims

The following source classes were excluded from verified factual records:

- **Shiksha, Careers360, Collegedunia, CollegeDekho, College Pravesh and coaching portals:** secondary aggregators; any values found there were not imported as official facts.
- **Unsourced placement package claims and social-media graphics:** excluded unless traceable to an official IIT Bombay report.
- **Copied cutoff tables:** excluded because the user-provided original verified cutoff dataset is authoritative.
- **Anonymous forum assertions about faculty, ragging, hostel quality, safety or placements:** retained only as limited qualitative evidence when useful; never used for official numbers.
- **Company-logo lists without a date:** excluded as proof of current recruiter participation.
- **Rumours, personal disputes and allegations naming private individuals:** excluded entirely.
- **Private groups, authenticated portals and inaccessible records:** not bypassed.

## Specific unsupported claims not imported

- Guaranteed branch change, scholarship, hostel room type, placement or package.
- A single definitive campus area while official sources conflict.
- A complete current recruiter list, because public company-level evidence is incomplete.
- Numeric ratings for mess, hostel, teaching, senior-junior culture or campus life.
